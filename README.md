Проект от ФЦТ МГУТУ
Разрабатывается студентами Факультета цифровых технологий МГУТУ им. К.Г. Разумовского в рамках подготовки к акселератору ["IT-стартапы для вузов" на Leader-ID](https://s3.objstor.cloud4u.com/unti-prod-people/file/presentation/project/0c1osvzjwv.pptx).

Чек-лист первого контрибьюта:
1. Сделайте форк репозитория (кнопка Fork в правом верхнем углу).
2. Клонируйте свой форк локально:
   git clone https://github.com/yaaankaa05-source/Ivanova-Yana-mgutu-fct
3. Перейдите в папку проекта:
   cd Ivanova-Yana-mgutu-fct
4. Установите зависимости:
   pip install -r requirements.txt
5. Запустите тесты:
   python -m pytest tests/
6. Внесите правки → зафиксируйте → отправьте PR.
