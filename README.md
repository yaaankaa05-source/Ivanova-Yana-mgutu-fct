# -Ivanova-Yana-mgutu-fct
Разработка интеллектуального карандаша для автоматизированного нанесения изображений на крупногабаритных объектах. Проект ФЦТ МГУТУ.  Программно-аппаратный комплекс включает промышленный AR-шлем, интеллектуальный карандаш с системой отслеживания движений и специализированное ПО.
